require() is the command used to import a module.

Example:
const express = require('express');

In order to require a file, one must write the absolute path, using ./ to indicate current directory.
For example:

app.js
  • console.log('here's the app.js');
index.js
  • require('./app.js'); // importing a file

index.js
  • require('express'); // importing a module
