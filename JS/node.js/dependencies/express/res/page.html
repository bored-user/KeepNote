res is the respose object.
It has methods like json(), to send json data or send() to send any type of data. One can send json using the send() method and giving a json object.

One can also render HTML pages using res.render(). It takes the file name (without extension) that will be rendered as the first argument and arguments to be given to that specific file as an object in the second parameter.
Example:
res.render('file_name', {arg: 'value'});

In Mustache:
{{arg}}

res can also redirect the user to another page: res.redirect('/');
