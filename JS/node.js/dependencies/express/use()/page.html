use() is a function used to direction routes.
It takes the route path as the first argument and the router itself as the 2nd.

One can set a helper object before the default route in order to pass default arguments to Mustache.
The default name for a helper file is helpers.js. One can export values using exports.key = 'value'.
The file is loaded using require() and passed to Mustache pages using:
app.use((req, res, next) => {
  • res.locals.h = require('./helpers.js');
  • next();
})
