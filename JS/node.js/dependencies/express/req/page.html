req is the first argument used in the function which is itself an argument of the get() function.
It represents the request and has some properties like query (for GET requests) - which represents the query strings (e.g ?age=1; req.query.age == 1) -, body (for POST requests) - req.body - , and params - which indicates that shortcut for getting info (e.g get('/posts/:id'); req.params.id will be the value of the id field).
