The set() method is used to set a attribute to an specific value.
For example, one can set the port using express().set('port', PORT_NUMBER).

set() is also used to set the view engine and its directory.
Example:
app.set('view engine', 'mst'); // where 'mst' is the file extension
app.set('views', `${__dirname}/views`); // and views is the html folder
