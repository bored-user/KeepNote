{{#loop_variable}}
<p>{{counter}}</p>
{{/loop_variable}}

See how the loop structure is the same as the conditional one. That's because Mustache internally analyses the given data type and adapts to a conditional (boolean type) or a for loop (array).


To make for loops:
res.render('file_name', { arr: [{ item: 'val1' }, { item: 'val2' }] });

In Mustache:
<ul>
{{#arr}}
<li>{{item1}}</li>
{{/arr}}


One can also interpret an array (which has no repeated keys) using:
{{#arr}}
{{.}}
{{/arr}}
