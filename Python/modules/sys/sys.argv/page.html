List containing given arguments.

The first item of the list (0) is always the file name (if the script is called using "python", sys.argv[0] is "app.py". If it's called by ./app.py, that's sys.argv[0]).
The other items are other given arguments.

Examples:
  • python3 app.py arg1 arg2 // ["app.py", "arg1", "arg2"]
  • ./app.py arg1 arg2 // ["./app.py", "arg1", "arg2"]
