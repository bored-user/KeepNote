Represent true or false. 1 or 0.

They're written in capita letter (the first one).

True
False

They can be also defined through a check.
Examples below:

myVar = 1 == 1 # True
myVar = 5 > 7 # False
myVar = (1==1) or (5>7) # True
myVar = (1==1) and (5>7) # False
myVar = not True # False
myVar = not 1==1 # False
myVar = not 5>7 # True
