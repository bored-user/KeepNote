Constraints

It's possible to constrain the generic type using the where clause:

class CustomList<T> where T : IComparable
// That means "<T> may be any value [that is] that implements IComparable interface"
{
	public void Add(T value)
	{}
	
	public T this[int idx]
	{
		get
		{}
	}
}

Note that in the example above, an interface was used as constraint. It's also possible to use a [class] (meaning “where <T> is of type [class]”), struct ("where <T> is a struct") and even the class keyword
