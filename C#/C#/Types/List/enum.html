enum

An enum is a set of related constants in form of key/value pairs.

An enum is defined outside any method (on the class root). It's values can be byte, short, int or long (and their and their signed/unsigned versions).

 

 

 

Note the : int. That's how one can define the enum value's type. Also, it's not necessary to define the enum key values (if no value is set, the fist will be zero and the others will be incremented by one), but it is, however, a good practice to do it.
