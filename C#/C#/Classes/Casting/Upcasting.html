Upcasting

Upcasting is the process of casting an object of a child class (derived class) into a parent/ base class.

One can easily done this like:

class ParentClass
{}

class ChildClass : ParentClass
{}

class Program
{
	static void Main()
	{
		ChildClass cc = new ChildClass();
		ParentClass pc = cc;
		// `pc` is now a casted ParentClass
	}
}
