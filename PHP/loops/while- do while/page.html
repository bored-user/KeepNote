The while loop execute the commands given while the given condition is true.
One should use some kind of exit condition.

while (true) will execute forever.

Examples:

$myVar = 0
while ($myVar <= 10) {
  • commands;
  • $myVar++
}

On the first loop, myVar will be 0. Then it'll turn to 1. And 2. And 3... And 9. And finally 10, exitting the while loop (since the condition is false).

The do while will first execute and only then check the condition.
It'll certainly execute at least once.
