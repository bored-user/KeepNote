A function are blocks of code that are reused.
The naming rules are the same as variables.

A function is defined using
  • function FunctionName($par1, $par2) {
    • commands
  • }
 and called using
  • FunctionName(1, 2)
The variables set between parentheses are arguments (or parameters). They are set to the value sent when the function is called.
You can set before using or using before setting. As long as they are set in your code it's all good to go.
