Anonymous classes are classes that are deleted just after they're used.
They should be used when the class the programmer is creating does something very basic (or does only one thing, once), so that it isn't necessary to create a whole class just for that.

In order to use an anonymous class, one must use:
$obj = new class() {
  • public function myFunc() {}...
}

Those created properties and methods can be accessed using the $obj object, just like in a normal class/ object.

Quick note: if there's no constructor (__construct method) there's no use to the brackets after the class keyword.
