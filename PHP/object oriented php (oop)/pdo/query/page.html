One should first set the SQL query and then use it on a $stmt variable, like:
$stmt = $this->connect()->query($sql); // where $this referes to the connect function set on the database class which was extended to the currently working class.
And then use a $stmt->fetch() or $stmt->fetchAll() to get the result accordingly.

Using prepared statements (when there's user input):
$stmt = $this->connect()->prepare($sql);
$stmt->execute(["item 1", "item 2"]); // in this case, the SQL query will have some "blanks" that need to be filled. Every time there's a blank, one should use the question mark as a placeholder. Example: $sql = "SELECT * FROM users WHERE name = ? AND id = ?"; The first array item will indicate the value of the name row, while the second item will indicate the value of the id row, and so on.

This will also works on INSERT queries (need to use prepared statements for security).
