Static means it doesn't change, just like constants.
It is useful when setting a variable which doesn't deppends on users.

It can be accessed without instanciating the object.


Example:

class Example {
  • public static $name = "Joao";
}
echo Example::$name // When using statics, we use :: (scope resolution operator) instead of ->
// One can also use Example::myStaticFunction()

class Example2 {
public static function Example3() {
  • static $example = 123
  • echo self::$example; // Echoes 123. We use self instead of $this when using static.
}

When using "$this->" it ins't necessary to use the dollar sign. However, when using a static property, one must use "ClassName::$prop".

Finally, to run a static function (Example3, in this case), on must use "Example2::Example3()".
