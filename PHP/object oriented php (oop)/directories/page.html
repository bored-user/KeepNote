Included and required files should go into a folder named "includes" (on root directory), and should be named file.inc.php
Class files should go into a "classes" folder, also on root dir, and should be named ClassName.class.php

If one is going to use namespaces it is necessary to add a "NamespaceName" folder, inside the class folder, to allow the spl_autoload_register() to run correctly.
