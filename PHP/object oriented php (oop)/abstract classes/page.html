  • abstract/abstractName.abstract.php
They're used to define a basic skeleton for classes to derive from. Classes can be extended from them, using "extends".
All abstract methods defined inside an abstract class must be declared inside the extending class as well (with the same parameters, if any).

In order to declare an abstract class, we use the "abstract" followed by the "class" keyword.
"abstract [visibility] function" is used to declare abstract methods (only inside abstract classes).
