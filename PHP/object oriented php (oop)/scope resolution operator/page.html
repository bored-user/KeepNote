The SRO is the double colons ("::").
It is used to access static properties and methods from within or without their parent classes.

To access a static property from that specific class from within it, use "self::$prop" (with the dollar sign).
To access a static inherited property, use "parent::$prop" (with the dollar sign).
To access a property from within the class (not static), use $this->variableName (no dollar sign).
