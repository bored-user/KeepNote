$this is used to access properties or methods from within the same class it was called originally.
Example:

Class MyClass {
  • private $myProp1 = "hello";
  • private $myProp2 = $this->myProp1;
}
The private property "myProp2" has the same value as "myProp1", since we used the $this reference.
Notice that when assigning the value of "myProp2" to "myProp1", I didn't use the dolar sign. That's because properties and methods don't need the dolar sign on referencing (only when declaring).


$this is also used on extending.
Example:

Class MyClass {
  • private $myProp1 = "hello";
}

Class MyClass2 extends myClass {
  • private $myProp2 = $this->myProp1;
}
