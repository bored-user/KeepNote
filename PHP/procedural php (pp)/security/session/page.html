A session is used to store data.
It ends as soon as the user closes the browser (wiping all information).

start_session() is used to start a session.
session_unset() is used to unset session data, clearing all variables.
session_destroy() is used to destroy the data, deleting all data, including possible files.

To store data in a session it's used $_SESSION["key"] = "value".

In order to share session data among multiple pages on your website, one must use session_start() in each page.
