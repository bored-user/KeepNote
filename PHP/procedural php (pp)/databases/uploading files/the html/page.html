<input>: There's an input type named "file". It is used to generate the file-selection dialog in order to upload a file. One can also upload multiple files specifying the "multiple" attribute and using brackets on the input name attribute, specifying it is an array.

<form>: There's an attribute of forms named "enctype", which means encoding type. It is usually set to "multipart/form-data" in order to upload files.
