Arrays are a variable set.
They can be either a nominated list or a numerated list.
Arrays (unless set otherwise) ALWAYS start at the undes 0

Arrays are defined using the built-in array function.

$myArray = array(1, 2, 3, 4, 5)
echo $myArray[0] => prints 1

$myArray = array(
        • 1 => "value 1",
        • 2 => "value 2",
        • 3 => "value 3"
      • )
echo $myArray[1] => prints "value 1"
