One can set a string to the name of a variable and call it using (), giving it arguments or not.

Example:
function myFunc($val) { echo $val; }
myFunc("hello world!") // echoes "hello world!"
$str = "myFunc";
$str("hello world!") // also echoes "hello world!"


One can also create anonymous function like in JS.
$myVar = function() { return "hello" }
echo $myVar // echoes "hello"
