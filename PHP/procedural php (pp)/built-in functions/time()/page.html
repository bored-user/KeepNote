time() function returns the current time measured in seconds since UNIX EPOCH (01/01/1970 00:00:00 GMT).

You can use time() + (7*24*60*60) to calculate next week seconds amount.
This can also be done using strtotime("+ 1 week"), which adds a week. Samething with + 1 day, + 1 year, - 1 year, - 1 month etc

Local time as string can be get by strftime() and local timestamp (if ever needed) by mktime().
