Takes a file and returns it's parent directory.
Example: dirname("/etc/something/somefile.txt") returns "/etc/something".

PHP 7 added the levels optional parameter, which allows the programmer to choose the number of directories up they wanna go. Example: dirname("/etc/something/somefile.txt", 2) returns "/etc".
Default value for levels parameter: 1.

One can use dirname(__FILE__) to get the directory name of the file they're working on (or simply use getcwd() function or __DIR__ magic constant).
