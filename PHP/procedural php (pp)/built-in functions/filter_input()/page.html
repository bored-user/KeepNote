Function taking a constant (the variable with the data. Example INPUT_POST and INPUT_GET), the index (the index where the data on the variable specified is located) and the filter type:
  • Validation filters: FILTER_VALIDATE_INT, FILTER_VALIDATE_EMAIL, FILTER_VALIDATE_URL, FILTER_VALIDATE_IP
  • Sanitize filters: FILTER_SANITIZE_SPECIAL_CHARS, FILTER_SANITIZE_INT, FILTER_SANITIZE_EMAIL, FILTER_SANITIZE_URL






Validate filters return a boolean (if the input is or isn't valid).

Sanitize filters returns the string sanitized.
Example:
  • FILTER_SANITIZE_SPECIAL_CHARS: Remove special characters like HTML markations.

FILTER_SANITIZE_EMAIL: Removes all characters which aren't allowed in email addresses.
