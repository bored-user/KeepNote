The rand() built-in function generates a random number between the parameters given, including both.
The first parameter is the minimun and the last is the maximun.

If only one parameter is given PHP consider it as being the maximun and the minimun is set to 0.
If no parameter is given, it generates a number between 0 and the maximun random number PHP is able to handle (one can use getrandmax() to know such number. It changes from platform to platform).

No floating points at all.
