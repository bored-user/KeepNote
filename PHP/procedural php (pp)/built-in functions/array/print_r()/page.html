The print_r() built-in function does roughly the same thing as echo.
But it's used to print human readable information (e.g arrays' structure).

It is considered a function because it returns a value (0 or 1).
The echo and return commands, for example, aren't functions, since they do not return any value.

$myArray = array("val1", "val2", "val3")
print_r($myArray) results in
Array
(
  • [0] => "val1",
  • [1] => "val2",
  • [2] => "val3"
)
