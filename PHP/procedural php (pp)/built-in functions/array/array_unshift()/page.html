Adds a element to the beginning of an array.

$myArray = array("item1", "item2", "item3")
print_r(array_unshift($myArray, "item0)) // Array ([0] => "item0", [1] => "item1", [2] => "item2", [3] => "item3", [4] => "item4")

If only one element is being added to the array it's better to use $array[] = $var.
