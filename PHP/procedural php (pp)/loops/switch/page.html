The switch loop is the simplest loop possible.
It loops through a static set, executing commands for each case.

Syntax:
  • switch($myVar) {
    • case "value 1":
      • commands;
      • break;
    • case "value 2":
      • commands;
      • break;
    • case "value 3":
      • commands;
      • break;
    • default:
      • commands;
  • }
The break command means "exit loop".
The default condition MUST always be the LAST, since PHP will execute it as soon as it reads it.

The switch loop can be (and mostly is) used with numbers, instead of strings.
