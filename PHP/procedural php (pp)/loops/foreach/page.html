The foreach loop goes through every item in an array.

Syntax:

$myArray = array(1, 2, 3, 4, 5);
foreach ($myArray as $item) {
  • echo $item // 1, 2, 3, 4, 5
}

The variable after the "as" is set to that specific value in that specific position where the loop is going through.

One can also use foreach ($array as $index => $value); where $index is the specific array index and key is the specific array key.
