Conditionals are used to check something.

In PHP, it's syntax is:
if (condition) {
  • commands
}

The "condition" is very often a variable value.
The block check if the expression written inside parentheses is true. If one write if (true), it'll always be true.

All numbers, positive and negative, except for 0 are synonims of true.
One can use the excalmation mark to invert the result. if (!true) means if (false).

If a variable is used it doesn't necessarily needs a value to compare to. If the variable is set, one can use if ($myVar). If the variable isn't unset (or set to 0, or set to false) it'll result in a true value, entering the block and executing the commands.

There's also the equal signs.
To check a value we use two equal signs (==). That means "same content". So, 2 == "2". Three equal signs (===) means same content and type. 2 === "2" results' false.
