It is the only expression that takes three operands.
Is is used as a short for an if block.

Example:
echo $v === "value" ? "if it's true" : "if it's false"
This will echo "if it's true" if $v === "value" and "if it's false" otherwise.


One can also use the ternary operator when defining a variable. It analizes the value of the variable and sets it accordingly. It'll only set it to the 2nd value if the original value is a falsy one ("", [], null...)
$v = 2;
echo $v; // 2
$v = $v ? "true" : "false";
echo $v; // "true"

$v = false;
echo $v; // false
$v = $v ? "true" : "false";
echo $v; // "false"
