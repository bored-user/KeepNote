In Julia, one can make basic if statements and use the ternary operator.

if x == 1
  • println("x == 1");
else # one can also use elseif to make else if blocks
  • println("x != 1");
end

println(x == 1 ? "x == 1" : "x != 1");

In Julia, however, one can also use the && operator:
x == 1 && println("x == 1") # the && will only execute the second instruction if the first one is true
