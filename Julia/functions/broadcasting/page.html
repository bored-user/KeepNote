One can call a function using normal f() syntax or using broadcast syntax: f.();

What broadcast basically does is it treats arguments (usually array items) as different items.
For example:

sq = n -> n ^ 2 # sq takes n and returns n ^ 2
sq(2) # 4
sq([1, 2]) # ERROR: there's no "^" operation on arrays
sq.([1, 2]) # [1, 4] It performs the operation on each array item.
