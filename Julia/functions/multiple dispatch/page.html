The reason why Julia accepts one to redeclare a function that's already been declared it the fact that one can declare multiple functions with the same name (and same type) which takes different argument types.

Example:

printArg(Int::arg) = println("The argument $arg is an Int")
printArg(String::arg) = println("The argument $arg is a String")
printArg(Bool::arg) = println("The argument $arg is a boolean")
printArg(arg) = println("We don't know the data type of $(arg). Run typeof() and find out!")
