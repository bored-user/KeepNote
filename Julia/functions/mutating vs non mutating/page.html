Mutation functions are the ones which changes (or alters) their arguments.
By convention they're written using an exclamation mark at the end.

Examples of that are push!() and pop!().
There are many functions in Julia which have both a mutating and a non mutating version.

For example, the sort() function:
arr = [3, 2, 1];
println(sort(arr)) # [1, 2, 3]
println(arr) # [3, 2, 1]
sort!(arr)
println(arr) # [1, 2, 3]
