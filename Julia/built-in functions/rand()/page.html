Generates and returns a arbitralily dimentioned array.
Takes three arguments:
the range in which the random numbers will be sorted, like 1:10;
array dimension 1
array dimension 2

It is important to notice that one can make combinations of those arguments, omitting the others:
rand(1:10) # will generate and return a random number (integer) between 1 to 10
rand(10, 10) # will generate a 10 dimension array with values between 0 and 1
rand(1:10, 10, 10) # will generate a 10x10 array which values are from 1 to 10.
