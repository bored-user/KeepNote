Restricting usage

One can restrict the usage of attributes using the AttributeUsage attribute on the attribute class.

It's possible to use attributes on:

  • Assembly (namespace)
  • Class
  • Constructor
  • Delegate
  • Enum
  • Event
  • Field
  • GenericParameter
  • Interface
  • Method
  • Module
  • Parameter
  • Property
  • ReturnValue
  • Struct

 

The syntax for restricting the use of attributes is as follows:

// the CoolAttribute can only be used on classes and structs
[AttributeUsage(AttributeTargets.Class | AttributeTargets.Struct)]
class CoolAttribute : Attribute
{}

 

Finally, notice that attributes are inherited by default. One can change this by adding the Inherited argument on the AttributeUsage attribute, using named arguments, like so:

[AttributeUsage(Inherited = false)]
class CoolAttribute : Attribute
{}

Also, be aware that it's possible to limit the usage of attributes to, for instance, only once, using:

[AttributeUsage(AllowMultiple = false)]
class CoolAttribute : Attribute
{}
