The <img> tag is used to insert an image into the webpage.
It has three main attributes: src (source), alt (alternative) and title.

The src specifies the source of the image; where it'll be loaded from.
The alt is a mandatory attribute (by the syntax rules of W3C). It specifies an alternative text to be displayed if the image isn't avalilable.
The title specifies the image title. It creates that beautiful placeholder shown when the mouse is kept on top of the image for some seconds.
