The <a> tag specifies anchors.
It is used to insert hyperlinks on webpage.

It has a main attribute: href. href stands for hyperlink reference and specifies the adress of the hyperlink.

It has other important attribute: target. target is used to specify the target of the weblink (_blank, _self, _parent or _top).
The _self is the default value. _blank means to open on a new tab.

A final attribute: download. It has no value and specifies to the server that the document should be downloaded, not opened. Often used to download pdf files.
