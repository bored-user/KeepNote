There are two types of HTML tags:
The individual ones and the pair ones.

The individual ones (e.g <meta>, <img>, <input>) only opens. It does not close.
The pair ones have the opening tag, <tag> and the closing one, </tag>.


Every HTML document starts with the doctype tag. It is an individual tag to specifiy to the browser that that's an HTML doc.
<!DOCTYPE html>
Followed by a html pair tag (<html></html>), a head tag (<head></head>) and a body (<body></body>).
Both body and head tags are located inside the html tag.
