The link tag does not specify hyperlinks.
Instead it manages external link references.

It has normally two attributes: rel and href.
The rel attribute specifices the relationship between the resource and the page.
Href means hyperlink reference. It's the path to the resource.

When importing a stylesheet (CSS) there's also another attribute: type.
The type specifies the resource type. When importing CSS, it's text/css.
