p5.js works in a very specific way: it sets the style of a shape BEFORE drawing it.
That means that if one wants to draw a green rectangle, they should use the fill("green") function before the actual rect() function.

This means that every style function (such as fill(), noFill(), stroke(), strokeWeight(), etc) should be placed before the actual drawing functions (such as rect(), triangle(), arc(), ellipse(), quad(), etc).
