The way to align things (shapes).

The default value is CORNER. This means that if you place a rectangle at the "center" of the screen but it's rectMode() is set to CORNER (or not set at all since it's the default value), it WON'T BE at the center of the screen. IT'S TOP-LEFT CORNER WILL.

So, in order to solve that, one should use the rectMode(CENTER) (to have the alignment calculated relative to the rectangle's center) before drawing the rectangle itself.
