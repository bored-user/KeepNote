It's also possible to make a classes' method return self, to allow chaining.

Example:

class MyClass
  • def getSelf()
    • return self
  • end
  • def sum(a, b)
    • return a + b
  • end
end

my_class = MyClass.new()
puts my_class.getSelf().sum(2, 2) # 4. my_class.getSelf() returned the class itself (same as the value of my_class), which is where I run the sum() method.
