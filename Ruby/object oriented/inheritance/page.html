To indicate inheritance in Ruby, one must use the < sign.


class MySuperClass
  • def sayhello()
    • return "hello"
  • end
end

class MyClass < MySuperClass
  • def sayhi()
    • return "hi"
  • end
end

my_class = MyClass.new
puts my_class.sayhello # "hello". It can access it even though the method wasn't set on the class itself.
