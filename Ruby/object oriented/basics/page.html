In Ruby, one can create a class using the class keyword.


To set properties inside a class, one can use the attr_accessor function followed by a colon and the attribute name, like so: attr_accessor :my_attr1, :my_attr2

To  set methods, one can use the def keyword, creating a function like we normally do on a global scope.
