Unless blocks are kind of the oposite of if blocks: it only runs the commands inside it if the specified condition evaluates to false.


For example:

# Using if block
num = 1
if num == 1
  • puts "num == 1"
else
  • puts "num != 1"
end

# Using unless block
num = 1
unless num = 1
  • puts "num != 1"
else
  • puts "num == 1"
