Exception handling can be done using the begin/ rescue blocks:

begin
  • num = 10/0
rescue
  • puts "Some error occured"

If one needs to catch an specific error:

begin
  • num = 10/0
rescue ZeroDivisionError
  • puts "zero division error"
It's even possible to create multiple catch blocks for multiple exceptions:

begin
  • num = 10/0
rescue ZeroDivisionError
  • puts "zero division error"
rescue
  • puts "some other random error"
Finally, one can also raise an error using the raise command:

raise "My custom exception!"
