One can perform usual arithmetic operations:

Examples:
puts 3 + 2 # 5
puts 3 - 2 # 1
puts 2 / 2 # 1
puts 3 * 2 # 6
puts 3 ** 2 # 9
puts 9 % 3 # 0

Obs.: When performing mathematical operations on numbers, the result might be rounded:
puts 10 / 3 # 3
puts 10 / 3.0 # 3.33333333

Incrementing and decrementing also works:
puts 2 += 2 # 4
puts 2 -= 2 # 0

abs and round functions can run directly on numbers:
puts -4.abs() # 4
puts 4.5.round() # 5

And finally, the Math class works just like in JavaScript:
puts Math.sqrt(9) # 3
puts Math.log(100) # 2
