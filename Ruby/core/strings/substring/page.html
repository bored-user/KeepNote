One can retrieve a substring using squre brackets.
It's possible to use both positive (left-right) and negative (right-left) indexes.

By default, if a second index (the step count number) isn't specified, Ruby consider 1.
The second index doesn't work when using negative numbers: 'Hello'[-1] returns the same as 'Hello'[-1, 3]

Syntax:
puts 'Hello'[0] # 'H' (which is the same as puts 'Hello'[0, 1])
puts 'Hello'[-1] # 'o'
puts 'Hello'[0, 1] # 'He'
