In order to access a property/ method of a class within itself, we use the self keyword:

class MyClass
  • attr_accessor :a
  • def myMethod()
    • puts self.a
  • end
end

my_class = MyClass.new()
my_class.a = 123
my_class.myMethod() # 123

It's also possible to access the classname through the self keyword using the self.class property.


One can also access inner properties of a class from within it using the @ symbol:

class MyClass
  • attr_accessor :a
  • def myMethod(a)
    • @a = a
    • puts @a
  • end
end

my_class = MyClass.new()
my_class.myMethod(123) # 123


The difference between using @ and self is that @ can only be used to refer to properties & @ will modify the value instead of executing a possible getter/ setter function for it; while self will execute the getter/ setter & can be used with methods.
