One can override the default constructor from a child class by creating a new initialize method.
It's also possible to call the constructor function from the super (parent) class using the super() function, like so:

class SuperClass
  • def initialize
    • puts "Initializing the super class!"
  • end
end

class ChildClass < SuperClass
  • def initialize
    • puts "Initalizing the child class!"
    • super()
  • end
end

child_class = ChildClass.new # "Initalizing the child class!" "Initializing the super class!"
