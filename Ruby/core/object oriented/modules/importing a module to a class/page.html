There are some ways to add a module to a class: include, prepend and extend.
The differences between them are:


include: When using include, the module methods are imported as instance methods. Which means that one needs to instantiate the class (MyClass.new) before using them.

extend: When using extend, module methods are imported as class methods. Therefore, there's no need to instantiate the class before using said methods (MyClass.MyModuleMethod).

prepend: Prepend is a lot like include. The only difference is that prepend overwrites the class methods if a name collision happen. Example: there's a run method on MyClass and a run method on MyModule. If one uses include MyModule on MyClass and runs MyClass.new.run, the class run function will be executed. If they decide to use prepend instead, the module run function will be executed.
