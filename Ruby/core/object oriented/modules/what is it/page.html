Modules are something kind of like classes. However, they cannot be instantiated.
Classes can include/ prepend/ extend modules.

So, modules are basically like interfaces in the sense that it's possible to use the methods of a module inside multiple classes through the use of the include/ prepend/ extend keywords.

However they're not like interfaces because interfaces cannot be instantiated nor have their methods used directly (without being 'interfaced' by a class).

So, in summary: A module may contain only methods. It's possible to call module methods directly (outside of classes: MyModule.my_method()). It's not possible to instantiate a module (MyModule.new). It's possible to add a module to a class which may or may not override the classes' inner methods (name collision).
