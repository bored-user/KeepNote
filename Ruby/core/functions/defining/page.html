To define a function, the "def" and "end" keywords are used (just like in Python, at least the def part). Function naming rules are the same as variable ones.

Example:
def sum(n1, n2)
  • return n1 + n2
end

puts sum(2, 2) # 4


One can set default values using the equal sign in function arguments definition:
def sum(n1, n2 = 0)
  • return n1 + n2 # if a second argument isn't given, the function will return the first one
end

puts sum(2) # 2
