Dictionaries are basically associative arrays (also called hashes).

In order to define a dictionary in Ruby, we use the arrow syntax:

dict = {
  • "key" => "value",
  • "key_2" => "value_2",
  • 3 => "value_3"
  • :key_4 => "value_4"
}

One can also set a dictionary key-value pair after said dictionary was set:

dict["key_5"] = "value_5"
