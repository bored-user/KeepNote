Prematurely ending loops

It's also possible to exit loops before the specified condition is met or even stop the current iteration and go to the next one using the break and next keywords.

The first will stop the loop completely and continue parsing the code from bellow it. The second stops the current iteration and continue running the loop (same effect as the continue keyword in most programming languages).

 

arr = %w[r u b y]
for char in arr
	next if char == 'b'
	puts char #=> 'r' 'u' 'y'
end

for char in arr
	break if char == 'b'
	puts char #=> 'r' 'u'
end
