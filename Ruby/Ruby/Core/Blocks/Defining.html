Defining

There are two ways to define a block:

  1. Using the Hash-like syntax ({})
  2. Using the do/end keywords

By convention, when there's only one line to be executed in said block, the Hash-like syntax should be used. Otherwise, do/end.

 

# Hash-like
{ |opt_arg| puts "inside block, #{opt_arg}" }

# do/end
do |opt_arg|
	puts "insde block, #{opt_arg}"
end
