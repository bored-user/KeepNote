write

write is a method that can be called on the File object. It writes content to a file.

File.write('file', 'content')

 

It can be also called on a File instance

f = File.new('file', 'r')
puts f.write('a')
f.close # write changes to file

 

The difference between both methods is that the first writes content directly, while the second will only write the contents to the file after the close method is called.
