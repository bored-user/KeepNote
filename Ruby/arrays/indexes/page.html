In order to get a single element at an specific position, the [index] notation is used: arr[0] # gets the first element

In order to get elements at specific positions, one can make the same they would when dealing with strings.
Example:
arr = ['a', 'b', 'c']
puts arr[0, 2] # start couting from index 0 (inclusive), get the next two elements => ['a', 'b']
# or with the dot notation:
puts arr[0..1] # get elements from index 0 (inclusive) to index 1 (inclusive) => ['a', 'b']
