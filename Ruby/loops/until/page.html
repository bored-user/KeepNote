The until loop is the opposite of the while loop: it executes a set of commands until a condition is met.


Example:

# While loop example
i = 0
while i < 5
  • puts i # 0 1 2 3 4
  • i += 1
end
# Until loop example
i = 0
until i > 4
  • puts i # 0 1 2 3 4. I used > 4 because 5 isn't greater than 5, so Ruby would also print 5 out
  • i += 1
end
