for loops are written starting with the for keyword, followed by the pointer variable, the in keyword, an iterable, the commands and the end keyword, like so

for i in 0..5
  • puts i # 0 1 2 3 4 5
end


One can also use the times() function:

5.times do |i|
  • puts i # 0 1 2 3 4 5
end

Or even, just like in Python or JavaScript:

arr = [0, 1, 2, 3, 4, 5]
for i in arr
  • puts i # 0 1 2 3 4 5
end


And there's also the each() method:

arr = [0, 1, 2, 3, 4, 5]
arr.each do |i|
  • puts i
end
